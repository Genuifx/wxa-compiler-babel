# wxa-compiler-babel
wxa babel compiler

# Install 
run `npm` to install. example: `npm i -D @wxa/compiler-babel`

# Usage
```javascript
// wxa.config.js
module.exports = {
    use: ['babel'],
    compiler: {
        babel: {
            // enter your babel option here
            // or you can put your configurations in .babelrc file.
        }
    }
}
```
